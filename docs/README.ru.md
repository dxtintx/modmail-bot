## Использование

Лёгкий ModMail-бот для Discord. Используйте это руководство для установки, настройки и запуска бота.

### Требования

-   Node.js 25+
-   Токен и ID клиента Discord-бота (из Discord Developer Portal)
-   Все intents должны быть включены

### Установка

1. Клонируйте репозиторий и перейдите в папку проекта:
    ```
    git clone https://github.com/dxtintx/modmail-bot.git
    cd modmail-bot
    ```
2. Установите зависимости:
    ```
    npm install
    ```

### Настройка

Создайте файл `.env` в корне проекта или используйте переменные окружения контейнера с как минимум такими переменными:

`TOKEN` — токен бота  
`CLIENT_ID` — ID клиента бота

### Запуск бота

```
npm start
```

### Права и роли

Все команды (кроме /faq) доступны пользователям, чьи ID находятся в whitelist.  
Только администратор сервера может управлять whitelist с помощью команд `/whitelist add` или `/whitelist remove`.

### Пример работы

1. Пользователь пишет боту в личные сообщения с проблемой.
2. Бот создаёт приватную почту, на которую могут отвечать пользователи из whitelist (модераторы).
3. Модераторы используют `/mails` для просмотра активных обращений.
4. Затем используют `/respond <id>` для ответа; сообщения пересылаются между модератором и пользователем.
5. После решения модератор использует `/close` для архивации обращения и сохранения переписки.
