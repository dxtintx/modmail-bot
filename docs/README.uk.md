## Використання

Легкий ModMail-бот для Discord. Скористайтеся цим посібником, щоб встановити, налаштувати та запустити бота.

### Вимоги

-   Node.js 25+
-   Токен бота Discord та його Client ID (з Discord Developer Portal)
-   Увімкнені всі інтенти

### Встановлення

1. Клонуйте репозиторій і перейдіть у папку проєкту:
    ```
    git clone https://github.com/dxtintx/modmail-bot.git
    cd modmail-bot
    ```
2. Встановіть залежності:
    ```
    npm install
    ```

### Конфігурація

Створіть файл `.env` в корені проєкту або задайте змінні середовища контейнера з принаймні такими змінними:

`TOKEN` — токен бота  
`CLIENT_ID` — Client ID бота

### Запуск бота

```
npm start
```

### Дозволи та ролі

Всі команди (oкрім `/how-to-use`) доступні користувачам, чиї ідентифікатори знаходяться в білому списку.  
Лише адміністратор сервера може керувати білим списком за допомогою команди `/whitelist`.

### Приклад роботи

1. Користувач надсилає боту повідомлення в DM з проблемою.
2. Бот створює приватний тікет, на який можуть відповідати користувачі з білого списку (модератори).
3. Модератори використовують `/mails`, щоб переглянути активні тікети.
4. Потім використовують `/respond <id>` для відповіді; повідомлення пересилаються між модератором і користувачем.
5. Коли питання вирішено, модератор використовує `/close` для архівації тікета та збереження транскрипта.
